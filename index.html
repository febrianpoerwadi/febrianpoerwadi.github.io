About Me

Halo! Perkenalkan nama saya adalah Febrian Dwinanda Poerwadi saya Lahir pada tanggal 20 Febuari 2007,Bandung dan Sekarang saya sudah kelas 10 SMA dan berskolah di SMAK3 Bina Bakti Bandung yang terletak di Jalan Taman Kopo Indah. Dan pastinya saya tinggal di Bandung,Jawa Barat.


Saya sudah suka dengan komputer sejak komputer saya masih memakai Pentium 4, Mulai SD kelas 5 baru mengenal akan Programming akan tetapi saya belum mengetahui apa-apa. Kemudian barulah SMP saya membuat website HTML pertama saya yang hanya bertulisan Hello World! dan saya juga belajar menggunakan visual basic untuk membuat Program. Kemudian sekarang saya mulai belajar kembali HTML. Saya belajar HTML sendiri dari Youtube!.
